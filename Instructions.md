# Portfolio Requirements: Minimalist Academic Style

## 1. Aesthetic Vision
- **Style:** LaTeX-inspired / Scientific Paper aesthetic.
- **Typography:** Use a Serif font stack (e.g., Computer Modern, Times New Roman, or Georgia) for the main body and a Monospace font for code/links.
- **Visuals:** ZERO images. Use horizontal rules (`<hr />`) to separate sections.
- **Theme:** Fixed Light Mode only. No dark mode.

## 2. Technical Requirements
- **Language:** HTML/CSS (Single File) or React + Tailwind CSS.
- **Responsiveness:** Must be readable on mobile but optimized for desktop "paper" width (max-width: 800px).

## 3. Page Structure
- **Header:** Name in large, bold Serif font. Subtitle with contact info (Email | GitHub | LinkedIn) separated by pipes.
- **Introduction** my introduction
- **Work Experience:** A text-based "Experience" section formatted like a CV.
- **Projects:** List of key repositories with descriptions and tech stacks.
- **Open Source:** A dedicated section for contributions to external libraries.
